# QR Code vers Sandbox

## URL de la Sandbox

L'URL de la sandbox Vercel suit ce format :
```
https://proto-verre-tms-git-sandbox-[username].vercel.app
```

## Comment trouver l'URL exacte

1. Connectez-vous à [Vercel Dashboard](https://vercel.com/dashboard)
2. Sélectionnez le projet **ProtoVerreTMS**
3. Allez dans l'onglet **Deployments**
4. Trouvez le déploiement de la branche `sandbox`
5. Copiez l'URL affichée

## Générer le QR Code

Une fois l'URL obtenue, générez le QR code avec :
- [QR Code Generator](https://www.qr-code-generator.com/)
- [QRCode Monkey](https://www.qrcode-monkey.com/)
- [QR Code](https://www.the-qrcode-generator.com/)

## Important

⚠️ Cette URL reste stable tant que la branche `sandbox` existe sur Vercel.

💡 Si vous supprimez et recréez le déploiement, l'URL restera la même tant que le nom de la branche ne change pas.
